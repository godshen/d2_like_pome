 - 语言: Python 3.9
 - 依赖库: qq-bot sdk，文档: https://bot.q.qq.com/wiki/develop/pythonsdk/
 - 云主机: GCP(Google Cloud Platform)， 文档: https://cloud.google.com/compute/docs/apis
 - 服务器: CentOS 7.9
 - 数据库: MySql 8.0.29
